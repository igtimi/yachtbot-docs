Water Current Commands

Commands for the Water Current software module are prefixed by "WCT ". This software module lets a user track in water and out of water points of a floating stick, to calculate water current movement.

ON:
Will allow the module to track user input on in and out points.
OFF:
Will turn off the module to track user input on in and out points.
MINDIST:
MINDIST" The module will not create a VDR NMEA string if the minumum distance traveled by the current is less than MINDIST. Default is 0.
MINTIME:
The module will not create a VDR NMEA string if the minumum time past between stick in and stick out is less than MINTIME. Default is 10 seconds.
SMN:
The module will not create a VDR NMEA string if the reported TRUE wind speed in the store is proportionaly low compared with the water current speed. This is an HPSZN special.
CONVENTION [FROM/TO]:
with FROM, 180 degress is added to the Water Current direction to say the current is "From" the direction. The correct nautical term is to use TO.
SOURCE [BUTTON/DISPLAY/BOTH]:
this command controls the source of user inputs for "Stick In" and "Stick out" events. The sources can be either, Power Button taps (BUTTON) or a physical button installed in the WindBot display (DISPLAY) that uses the GPIO pin inputs. If the required hardware isn't installed for DISPALY input on the WindBot IMU board, this module can spam the logs, change the SOURCE to BUTTON. BUTTON will allow Power Button taps "1 tap = In" "2 taps = Out".
