Global Commands

Prefix-less commands handled by the primary system.

UNIT_ID [n]:
0 <= n < 256. Lists or sets the unit ID.
UPTIME:
Reports how long the device has been running for.
INFO:
Lists all key general session information.
SERIAL_NUMBER [s]:
Lists or sets the unit serial number.
SERIAL_TAG [s]:
Lists or sets the unit serial tag.
VERSION:
Lists the hardware and firmware versions.
RESTART:
Restart the device immediately.
RESTART [t]:
Schedule restart after [t] seconds.
RESTART CANCEL:
Cancel a scheduled restart.
POWER OFF:
Powers off the device.
POWER FLASH <ON/OFF>:
Sets whether or not the power light should flash.
BATTERY_VOLTAGE:
Lists the current battery voltage. VOLTAGE is an alias.
TEMPERATURE:
Lists the current battery temperature.
CURRENT:
Lists the battery current.
ACCUMULATED CURRENT:
Lists the accumulated battery current. ACC_CURRENT is an alias.
ALERT VOLUME [value]:
Lists or sets the volume for alert tones. value is a real valued number in the 0 to 1.0 range.
RUN_LEVEL:
Lists the main systemm run level.
BATTERY SHUTDOWN [value]:
Lists or sets the shutdown threshold for the battery, in volts. value should be a real valued number from 0 to 5.0.
SYNC_CANCEL_ON_SHUTDOWN <ON/OFF>:
Sets whether or not to cancel the sync process when the unit wants to shut down.
TEST:
Runs test code.
HEARTBEAT_TIMER [value]:
Lets you choose the heartbeat timeout. It is not a CELL command, despite being most relevant to that module, because the heartbeat timeout lives in the global config. value should be a single precision floating point number of at least 1.0, representing the timeout in seconds. If no argument is given, it lists the current setting. The default is 60.0 seconds.
