Bluetooth Commands

Commands for the Bluetooth system. They are prefixed by "BT ". Optional arguments are denoted by square brackets with details below.

ON :
Turns the Bluetooth module on
OFF :
Turns the Bluetooth module off
MODE [NONE/RFCOMM/AUDIO] :
Switch to a restricted mode where the profile is restricted to a specific type. None means anything can connect. No argument prints out the current state.
MODE [NONE/RFCOMM/AUDIO] :
Sets the auth mode for SSP. Default is NONE (3). The e-ink display requires DISPLAY (0) No argument prints out the current state.
DISCONNECT :

Turn on or off Bluemix mode. Affects pairing and connections.

Disconnects the device that is currently connected

DISCONNECT :
Prints out the number of connected devices.
PAIR_AUDIO :
Attempts to pair an Audio Device
PAIR CLEAR [ALL/<address>] :
Removes the pairing of all devices, or a specific device.
SEND [command]:
Sends a command to the Bluetooth module. Commands of more that one word should be in "quote marks"
PASSKEY [CLEAR/<passkey>] :
Gets, sets or clears the Passkey (4 characters)
RESET :
Sends the reset command.
RESET :
Determines what incomming data the Bluetooth module will look for.
CONNECT_MESSAGE <ON>/OFF:
Turn on or off the text audio notifications for bluetooth. No argument prints out current value.
