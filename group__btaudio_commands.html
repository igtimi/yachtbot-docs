Bluetooth Audio Commands

Commands for the Bluetooth audio system. They are prefixed by "BT-HFP ".

START:
Starts audio recording. BT RECORD is an alias that needs to be deprecated. This is on by default
STOP:
Stops audio recording.
LIST_PAIR :
Lists all stored pairings.
MAX_CHANNELS:
Returns the maximum number of audio channels the BT can use, as set by the EEPROM config.
CHANNELS <n>:
For 0 <= n <= MAX_CHANNELS, this is the number of the channels the state engine should use. It requires restarting the device to take effect. Default is MAX_CHANNELS (normally 2)
LOOPBACK <ON/OFF> [LINK [MONO]]:
Enables or disables audio loopback. LINK swaps the left and right audio channels, turning two Bluetooth headsets into a communication channel. If MONO is specified, then when only one headset is available it will loopback to itself.
STOP:
Enables or diables beeps upon connection
